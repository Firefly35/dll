Function import_function

boost::dll::import_function

Synopsis

// In header: <boost/dll/import_function.hpp>


template<typename T> 
  boost::function< T > 
  import_function(const boost::filesystem::path & lib, 
                  boost::string_ref func_name, 
                  load_mode::type mode = load_mode::default_mode);
template<typename T> 
  boost::function< T > 
  import_function(const boost::shared_ptr< shared_library > & lib, 
                  boost::string_ref func_name);

Description

Returns boost::function<T> that holds an imported function from the loaded library and refcounts usage of the loaded shared library, so that it won't get unload until all copies of return value are not destroyed.

This call will succeed if call to boost::dll::shared_library::search_symbol(boost::string_ref ) function with the same symbol name returned true.

For importing function by its alias name use boost::dll::import_function_alias method.

Examples: 

boost::function<int(int)> f = import_function<int(int)>(
          boost::make_shared<shared_library>("test_lib.so"),
          "integer_func_name"
);

boost::function<int(int)> f = import_function<int(int)>("test_lib.so", "integer_func_name");

Parameters:

func_name

Null-terminated C or C++ mangled name of the function to import. Can handle std::string, char*, const char*.

lib

Path or shared pointer to library to load function from.

mode

An mode that will be used on library load.

Returns:

boost::function<T> that holds an imported function from the loaded library and refcounts usage of the loaded shared library.

Throws:

 boost::system::system_error if symbol does not exist or if the DLL/DSO was not loaded. Overload that accepts path also throws std::bad_alloc in case of insufficient memory.
